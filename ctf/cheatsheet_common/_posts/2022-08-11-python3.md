## HEX String -> Bytes
```python
s = '782079207a0a'
b = bytes.fromhex(s)
print(f'{s=}, {b=}')
```
```sh
s='782079207a0a', b=b'x y z\n'
```

## Bytes -> HEX String
```python
b = b'abc\n'
s = b.hex()
print(f'{b=}, {s=}')
```
```sh
b=b'abc\n', s='6162630a'
```
